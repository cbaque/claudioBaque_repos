## Installation

1. Clonar proyecto
2. ``` npm install ```
3. reemplazar ``` .env-template ``` por ``` .env ```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod

```
